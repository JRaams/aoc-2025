## 1. Requirements

- Nix (the package manager) https://nixos.org/download/

-or-

- Bun https://bun.sh/docs/installation

## 2. Running

`bun install`

Go to the folder of a specific day:

`cd src/01`

Create the file `input.txt` and paste aoc puzzle input

Run part A or B

`bun run a.ts`
