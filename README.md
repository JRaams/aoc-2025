# Advent of Code 2025

https://adventofcode.com/2025 by [Eric Wastl](http://was.tl/)

Solutions using TypeScript with Bun (v1.3.3)

## 1. Requirements

- Bun https://bun.sh/docs/installation

## 2. Running

`bun install`

Go to the folder of a specific day:

`cd src/01`

Create the file `input.txt` and paste aoc puzzle input

Run part A or B

`bun run a.ts`
